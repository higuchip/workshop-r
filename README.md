# Workshop: Introdução à Manipulação de Dados Florestais no R

> Do dado bruto de campo a tabelas organizadas — com aplicações em inventário florestal

## Sobre

Material do workshop de manipulação de dados florestais utilizando R e tidyverse. O workshop aborda desde a importação de dados de inventário florestal até a geração de tabelas resumo organizadas por parcela e por espécie.

**Instrutor:** Prof. Dr. Pedro Higuchi  
**Carga horária:** 8 horas (dia inteiro)  
**Pré-requisitos:** R e RStudio instalados

## Acesse o material

👉 **[Workshop Online](https://higuchip.github.io/workshop-r/)**

## Conteúdo programático

| Módulo | Conteúdo |
|--------|----------|
| 1 | Primeiros passos: importação e exploração de dados |
| 2 | Manipulação básica: pipe, select, filter, arrange |
| 3 | Transformação de dados: mutate e criação de variáveis |
| 4 | Agrupamento e resumo: group_by, summarise, count |
| 5 | Exercício integrador: processando um inventário completo |

## Dados

O arquivo `inventario.csv` contém 1.843 árvores medidas em 50 parcelas de Floresta Ombrófila Mista no planalto serrano de Santa Catarina, com as variáveis:

- **Parcela** — Número da parcela (1 a 50)
- **Especie** — Nome científico da espécie
- **DAP** — Diâmetro à altura do peito (cm)

## Como usar

1. Acesse o [workshop online](https://higuchip.github.io/workshop-r/)
2. Baixe o arquivo `inventario.csv` deste repositório
3. Abra o RStudio e crie um novo script
4. Siga as instruções do workshop, digitando os comandos no seu script
